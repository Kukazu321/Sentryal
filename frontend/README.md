Sentryal - Frontend

Next.js + TypeScript scaffold with TailwindCSS.

Dev:

```powershell
npm run dev
```

The app runs on port 3000.

Structure:

- /components - UI components (shadcn/ui will be integrated later)
- /hooks - custom React hooks
- /lib - api clients and helpers
- /styles - global css
- /types - shared TypeScript types
